Objetivo: 
--------------------------
Realizar sobre la aplicación Trello, una practica de automatización utilizando los conceptos adquiridos en los cursos de IntelliJ, Selenium Web Driver, TestNG.


Detalle de la practica:
--------------------------
Automatizar los escenarios:
- Iniciar sesión en Trello
- Crear un tablero
- Crear una lista
- Crear una card
- Cerrar sesión
- Tomar captura de pantalla cuando falle un test
- Parametrizar los valores requeridos para los test
- Configurar para que los test se puedan ejecutar con el navegador levantado o Headless
- Subir a Github el proyecto y contenga un README donde se describa el proyecto y la forma de ejecución

Nota: se requiere que la practica sea enfocada en el patrón Page Object Model


Forma de ejecución:
--------------------------
- Tener abierto el proyecto en el editor de Intellij
- Sobre el archivo "testng.xml" dar clic derecho
- Seleccionar la opción: Run '...\testng.xml'
